WebSIG Rabat-Salé-Kénitra

Système d'Information Géographique Web pour la gestion des ressources en eau et la cartographie du risque d'inondation dans la région Rabat-Salé-Kénitra.

Utilisation

  • Couches : bouton ☰ dans la barre d'outils (droite)
  • Statistiques : bouton 📊 dans la barre d'outils
  • Mesure : outils de mesure dans la barre d'outils
  • Recherche : bouton 🔍 pour géocoder un lieu
  • Popup : cliquez sur une entité pour ses attributs
